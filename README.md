课程介绍：

junit单元测试 注解就是代码里的 特殊标记，这些标记可以在编译、类加载、运行时被读取，并且执行相应的处理。通过注解程序员可以在不改变原有逻辑的情况下在源文件中嵌入一些补充信息。代码分析工具、开发工具和部署工具可以通过这些补充信息进行验证或进行部署。
